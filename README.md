jquery-color
============

### example

+ `$('#box1').color('red');      <!-- To set box1's color is 'red' -->`
+ `$('#box1').color();           <!-- To get box1's color -->`
+ `$('#box1').backcolor('red');  <!-- To set box1's backcolor is 'red' -->`
+ `$('#box1').backcolor();       <!-- To get box1's backcolor -->`
